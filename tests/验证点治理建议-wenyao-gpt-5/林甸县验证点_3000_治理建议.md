## 验证点名称: 林甸县验证点
面积: 1000亩
位于: 黑龙江省大庆市林甸县，黑龙江大庆
碱化度(ESP): 33.36%-57%
土壤pH值: 8.85-10.81
平均盐分含量: 7‰
盐碱成因: 土壤母质 灌溉不当 地势低洼
灌溉水来源: 河流
全年平均降水量: 427.5mm
年均蒸发量: 1635mm
年有效积温: 2678℃
综合治理目标与适配性
- 面积1000亩，土壤ESP为33.36%–57%，pH 8.85–10.81，平均含盐量约7‰，地势低洼且蒸发远大于降水，适宜采取“控水控盐+分区管控+化学改良+农林措施协同”的组合策略。
- 在不超过3000元/亩的预算前提下，优先解决地下水位与地表返盐问题，在耕作层实现“明显降盐+显著降pH+抑制返盐”，并兼顾农业生产与生态安全。

分区实施建议与预期效果
- 分区管控与生态廊道
  实施建议：
  - 依据高程、坡度、植被覆盖、质地及沟渠/水域分布，识别低阻力连通区，沿生态廊道优先布置灌排洗盐，禁止重点防治区内大规模建设开发；一般防治区推广耐盐作物或在适宜地块发展改良水稻；风险监督/预防区适度开发，优化配置。
  预期效果：
  - 在重点防治核心区以较低成本实现最大防治效益，洗盐效率高并可防止盐分向下游湿地扩散；一般防治区通过耐盐作物或改良水稻实现稳产增效（已验证的改良水田试验显示水稻产量水平可达较高水平）。

- 深浅井结合+排咸补淡+渠道防渗
  实施建议：
  - 浅井（含虹吸井、真空井）抽排浅层咸水控制地下水位；深井抽取碱性淡水，与浅层水咸淡混灌；汛期利用明沟强化伏雨洗盐，秋冬早春持续抽排浅层咸水、降低地下水位，随后用河渠淡水或井水进行冬春补淡；渠道实施防渗以减少渗漏补给。
  预期效果：
  - 排盐控水、优化水源、增强自然降盐能力，防止地下水位再次抬高及周边盐渍化和沼泽化；浅层咸水淡化，部分井位可出现明显淡化，农田稳产面积扩大。

- 咸水结冰灌溉（SAR=5、矿化度10 g/L、135 mm入渗）
  实施建议：
  - 在淡水短缺且土壤含盐量高的地块，采用结冰后自然融化入渗方式，优先用于耕作层盐分高的田块，结合后续排水与洗盐。
  预期效果：
  - 0–10 cm土壤平均含盐量降至1.1–1.2 g/kg；0–25 cm土层平均含盐量为2.9 g/kg，显著优于咸水直接入渗（10.6 g/kg），脱盐深度优于淡水与咸水直接入渗处理。

- 酸雨石（300 kg/亩，重耙20 cm×2）
  实施建议：
  - 在高pH、高盐苏打型盐碱土的耕地均匀撒施酸雨石，随即进行两次重耙至20 cm，持续实施并配合灌排体系维持长期效果。
  预期效果：
  - 4年期内土壤盐分与pH显著下降：典型地块盐分可由6.56 g/kg降至1.03 g/kg、pH由8.6降至7.2；亦可在pH 10.3的地块将pH降至约7.6，盐分降至0.45 g/kg；短期吸盐速率快、长期吸盐量大、解碱能力强且持续性好。

- 平整土地与深翻改土
  实施建议：
  - 对易积水与表层板结地块进行平整与深翻，配合有机质输入与后续洗盐，有利于入渗与团聚体形成。
  预期效果：
  - 改善土壤物理结构与耕作条件，配合综合措施提升地力与生产稳定性。

- 防护林带营造（沟沿2–3行；沟坡种草；沟底打柳木椿）
  实施建议：
  - 沿骨干沟、斗沟成带营造，形成防护与生物排水系统，沟坡草护、沟底加固。
  预期效果：
  - 减弱地面蒸发，树根吸收地下水形成生物排水，配合工程排水抑制返盐，改善田间小气候与土壤结构。

- 将盐分排入口与湿地水源分离
  实施建议：
  - 改造出水口并分流，保证排盐水与沼泽湿地水源分离，防止污染与二次盐渍化。
  预期效果：
  - 避免下游湿地受盐分侵扰，兼顾生态与农业用水安全。

- 水盐动态监测网
  实施建议：
  - 设置观察井与土壤定位点，定期观测水位、EC、pH及盐分，按监测结果动态调整排灌与洗盐频次。
  预期效果：
  - 及时掌握水盐演变，提升治理的针对性与效率。

- 调整作物布局与稻麦轮作；渠系连片种植
  实施建议：
  - 实施水稻-水稻-小麦轮作，逐步将高滩地转为旱作；在渠系内连片种植同一种作物，降低输水时间与渗漏。
  预期效果：
  - 降低灌溉渗漏补给与地下水位抬升风险，抑制水盐上行并提升用水效率。

- 一般防治区种植耐盐作物
  实施建议：
  - 在中轻度盐渍地推广耐盐作物或品种，结合分区管控降低改良投入。
  预期效果：
  - 成本较低、稳产增效，与工程与化学改良协同提高综合效益。

- 风险监督/预防区适度开发与优化配置；禁止大规模建设
  实施建议：
  - 加强规划监管与风险预防；重点防治区内禁止大规模建设，保留廊道通畅。
  预期效果：
  - 降低治理成本与生态风险，保障洗盐空间与水流通道。

综合效益与不选项说明
- 选项组合以“控水控盐”为核心：深浅井+排咸补淡+生态廊道洗盐+防护林与防渗，解决地下水位高与返盐的根因；辅以“化学改良+定向脱盐”：酸雨石快速降pH与吸盐，咸水结冰灌溉高效脱除耕作层盐分；配合“分区管控+作物布局”提升农业稳产与生态安全。
- 在预算约束下，本轮不采用五级排水系统（2800元/亩）或暗管网络（3800元/亩）等高投入工程，以避免超出3000元/亩；若后续资金充裕，可在重点核心区逐步加密工程排水以进一步提升长期稳态。

实施顺序建议
- 第一阶段：沿生态廊道布设洗盐与分流；深浅井布置到位；渠道防渗与防护林带同步推进；建立监测网；平整与深翻。
- 第二阶段：酸雨石改良与重耙；排咸补淡启动；根据监测开展定期洗盐。
- 第三阶段：咸水结冰灌溉用于耕作层高盐地块；落实作物轮作与耐盐作物布局；持续监测与调整。
