面积: 1352亩
位于: 甘肃省白银市景泰县，扬黄灌区
碱化度(ESP): 5%-20%
土壤pH值: 8.48
平均盐分含量: 15.5‰
盐碱成因: 土壤母质、灌溉不当、地势低洼、其他（红粘盘不透水层）
灌溉水来源: 河流、提灌
全年平均降水量: 185mm
年均蒸发量: 3038mm
年有效积温: 3038℃
### 景泰县盐碱地综合治理方案

#### 一、土地概况分析  
景泰县盐碱地面积1352亩，属扬黄灌区，土壤pH 8.48，盐分含量15.5‰，碱化度5%-20%。主要成因包括灌溉不当、地势低洼及红粘盘不透水层，年降水量仅185mm，蒸发量高达3038mm。需优先解决排水不畅、盐分表聚及土壤结构改良问题。

#### 二、治理措施组合与成本分配（总预算135.2万元）  
1. **排水系统建设（核心措施）**  
   - **措施**：开挖骨干排沟与农级明渠（深度1.9-2.8m），配套暗管排水（PVC波纹管），控制地下水位至临界深度以下。  
   - **成本**：骨干排沟和暗管排水（3500元/亩×200亩=70万元） + 农级明渠（1000元/亩×200亩=20万元） = **90万元**  
   - **效果**：降低地下水位1.5m以上，阻断盐分毛细上升，0-30cm土层脱盐率预期达40%-50%。  

2. **生物改良（低成本高效）**  
   - **措施**：种植耐盐草木樨（2年周期），配合秸秆还田（800元/亩）及有机肥施用（1500元/亩）。  
   - **成本**：草木樨种植（1000元/亩×500亩=50万元） + 秸秆还田（800元/亩×500亩=40万元） + 有机肥（1500元/亩×300亩=45万元） = **135万元**  
   - **效果**：草木樨根系改善透水性，2年后脱盐率76.3%；有机质提升30%-40%，pH降低0.3-0.5。  

3. **节水控盐（辅助措施）**  
   - **措施**：垄膜沟灌技术替代漫灌，减少渗漏。  
   - **成本**：节水灌溉技术（2500元/亩×100亩=25万元）  
   - **效果**：节水30%，表层盐分累积减少0.1%-0.2%。  

4. **局部客土（重度盐斑区）**  
   - **措施**：移除10-15cm高盐表土（50亩）。  
   - **成本**：客土移盐（5000元/亩×50亩=25万元）  
   - **效果**：表层盐分直接降低80%-90%。  

#### 三、预算分配与优先级  
- **优先级1**：排水系统（90万元） + 草木樨种植（50万元） = 140万元（超支部分通过缩减客土面积调整）。  
- **优先级2**：秸秆还田与有机肥（85万元） + 节水灌溉（25万元） = 110万元（分2年实施）。  

#### 四、预期综合效果  
- **短期（1-2年）**：地下水位下降1m，盐斑区面积减少50%，草木樨覆盖抑制蒸发返盐。  
- **长期（3-5年）**：土壤含盐量降至3‰以下，pH稳定8.0左右，可种植枸杞或耐盐小麦，亩产值提升至3000元以上。  

#### 五、实施建议  
1. **分阶段推进**：首年完成排水工程与草木樨种植，次年追加有机肥和节水设施。  
2. **监测调整**：每季度监测地下水位及0-50cm土层盐分，动态优化灌溉量。  

本方案以排水控盐为基础、生物改良为核心，兼顾成本与可持续性，符合景泰县干旱高蒸发环境特点。
